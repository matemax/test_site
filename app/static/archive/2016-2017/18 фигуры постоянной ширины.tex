\begin{definition}
Шириной выпуклой  фигуры в данном направлении называется расстояние между парой опорных прямых, перпендикулярных данному направлению.
\end{definition}


\begin{definition}
Фигурой постоянной ширины называется выпуклая замкнутая фигура для которой ширина по каждому направлению одна и та же. Её называют диаметром данной фигуры, будем обозначать её $d$. Фигуру постоянной ширины будем обозначать далее $F$.
\end{definition}

\begin{task}[Теорема Барбье] Периметр фигуры постоянной ширины с диаметром $d$ равен $\pi d$ 
\end{task}


\begin{task}
\sub Пусть $A,B\in F$, тогда $AB \leqslant d$.
\sub Для каждой точки $A$ на границе  $F$ есть точка $B\in F$ такая, что $ AB=d$.
\end{task}

\begin{task}
Фигура постостояной ширины является центрально симметричной тогда и только тогда, когда она является окружностью.
\end{task}

\begin{definition}
Описанной окружностью фигуры называется окружность с наименьшим радиусом $R$, содержащая всю фигуру.
\end{definition}

\begin{definition}
Вписанной окружностью фигуры  называется окружность наибольшего радиуса $r$, содержащаяся полностью в фигуре.
\end{definition}

\begin{fact}
Для любой ограниченной фигуры  существует единственная описанная окружность. Для вписанной окружности это неверно в общем случае.
\end{fact}

\begin{task}
Для фигуры постоянной ширины центры вписанной и описанной окружности совпадают, причём $d=R+r$.
\end{task}

\end{task}
